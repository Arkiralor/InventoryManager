# Inventory Manager - Backend

This repository contains the backend code for the Inventory Manager application, built using Django and Django REST Framework. It provides APIs for user authentication, inventory management, and other related functionalities.

## Features
- User registration and authentication
- JWT token generation and management
- Inventory item CRUD operations
- Role-based access control
- Comprehensive logging and error handling
